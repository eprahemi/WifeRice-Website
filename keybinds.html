Default Keybindings

Essential keybinds to get started. Super key = Windows key.

KeybindAction
Super + WOpen wallpaper picker
Super + HOpen guide (keybinds, system info, changelog)
Super + DOpen app launcher
Super + BOpen battery popup
Super + NOpen network manager
Super + XOpen volume popup
Super + SOpen calendar/weather
Super + COpen clipboard manager
Super + POpen movies/humor popup
Super + MOpen monitor manager
Super + Shift + SOpen settings panel
Super + Shift + NOpen Quick Notes
Super + Shift + WOpen workspace overview
Super + Shift + MOpen system monitor
Super + Shift + TOpen focus timer
Alt + QOpen music player
Super + TOpen terminal (Kitty)
Super + GOpen Google Chrome
Super + JOpen Microsoft Edge
Super + EOpen file manager (Thunar)
Super + FOpen Spotify
Super + AOpen Discord
Super + KOpen Steam (optional — not installed by default)
Alt + VOpen VS Code
Super + VOpen PulseAudio volume control (pavucontrol)
Super + QKill focused window
Super + Shift + FToggle floating window
Super + Shift + arrowsResize active window
Super + Ctrl + arrowsMove window in direction
Super + arrowsMove focus between windows
Super + LLock screen
Super + SpacePlay/Pause media (playerctl)
Super + RReload QuickShell UI
Super + 1–0Switch to workspace 1–10
Super + Shift + 1–0Move window to workspace 1–10
Super + TabFocus next monitor
Ctrl + LeftSwitch to previous workspace
Ctrl + RightSwitch to next workspace
PrintTake screenshot (region select, skip editor)
Shift + PrintTake screenshot (region select + editor)
Super + PrintTake screenshot (full screen)
Super + Shift + PrintTake screenshot (full screen + edit)

Tip: You can customize all keybindings in ~/.config/hypr/settings.json under the keybinds array. Custom binds override defaults.
